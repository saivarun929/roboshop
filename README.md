# Roboshop

Roboshop is a microservices based e-commerce application.

## Architecture
3-tier application with 11 components:
- MongoDB, Redis, MySQL, RabbitMQ (Databases)
- Catalogue, User, Cart, Shipping, Payment (Services)
- Frontend (UI)

## Deployment Approaches
- Manual - EC2 instances (current)

## Tech Stack
- Cloud: AWS (ap-south-1 Mumbai)
- OS: RHEL 9
